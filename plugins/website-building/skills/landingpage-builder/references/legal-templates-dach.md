### Zusätzliche Abschnitte je nach Dienst

| Dienst | Zusätzlicher Abschnitt nötig |
|---|---|
| **KI-API (z.B. Anthropic, OpenAI)** | Drittlandtransfer, Art der verarbeiteten Daten, Zweck |
| **Zahlungsanbieter (Stripe, PayPal)** | Verarbeitete Zahlungsdaten, Rechtsgrundlage Art. 6(1)(b) |
| **Social Media Plugins** | Zwei-Klick-Lösung, Drittlandtransfer |
| **Google Fonts** | Drittlandtransfer oder lokales Hosting (empfohlen!) |
| **YouTube/Vimeo Embeds** | Zwei-Klick-Lösung, Datenweitergabe |
| **Chat-Widget** | Verarbeitete Daten, Speicherdauer, Anbieter |

### Google Fonts – DSGVO-Empfehlung

Google Fonts LOKAL einbinden statt über Google CDN. Begründung: Beim Laden über Google CDN wird die IP-Adresse des Besuchers an Google übermittelt – das ist ohne Einwilligung ein DSGVO-Verstoß (LG München I, 2022). Lösung:

```html
<!-- NICHT SO (Google CDN): -->
<!-- <link href="https://fonts.googleapis.com/css2?family=..." rel="stylesheet"> -->

<!-- SONDERN SO (lokal): -->
<style>
@font-face {
  font-family: 'Plus Jakarta Sans';
  src: url('/fonts/PlusJakartaSans-Regular.woff2') format('woff2');
  font-weight: 400;
  font-display: swap;
}
</style>
```


### Cookie-Banner-Regeln (DSGVO)

1. **Opt-in, nicht Opt-out**: Keine Cookies setzen bevor der Nutzer aktiv zustimmt
2. **Echte Wahlmöglichkeit**: "Ablehnen" muss genauso einfach sein wie "Akzeptieren"
3. **Kein Dark Pattern**: "Akzeptieren"-Button darf nicht visuell bevorzugt werden (umstritten, aber sicher so)
4. **Einstellungen änderbar**: Nutzer muss Einstellungen nachträglich ändern können
5. **Dokumentation**: Einwilligung muss nachweisbar gespeichert werden

## Hinweis an den Benutzer

Alle Templates in dieser Datei sind **Strukturvorlagen** und ersetzen keine individuelle Rechtsberatung. Empfehlung:

1. Templates mit eigenen Daten ausfüllen
2. Von einem auf IT-/Internetrecht spezialisierten Anwalt prüfen lassen
3. Alternativ: Generatoren wie eRecht24, IT-Recht Kanzlei oder Datenschutz-Guru nutzen
4. Regelmäßig aktualisieren (mindestens jährlich oder bei Dienst-Änderungen)
