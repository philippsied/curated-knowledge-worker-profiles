# Website-Building Plugin

Skills fuer den Aufbau von Websites – von der Domain-Namensfindung bis zur fertigen, conversion-optimierten Landingpage.

## Enthaltene Skills

| Skill | Beschreibung | Herkunft |
|-------|-------------|----------|
| **landingpage-builder** | Erstellt datengestuetzte, high-converting Landingpage-Konzepte auf Basis eines Lean Canvas als HTML/CSS/JS-Prototyp. Nutzt 10-Sektionen-Blueprint, psychologische Frameworks (AIDA, PAS, StoryBrand, Cialdini), CRO-Best-Practices, SEO, DACH-Rechtspflichten (Impressum, Datenschutz, AGB), Kontaktformular und Newsletter-Integration. | Philipp Sied |
| **domain-name-brainstormer** | Generiert kreative Domain-Namen-Ideen fuer ein Projekt und prueft die Verfuegbarkeit ueber mehrere TLDs (.com, .io, .dev, .ai u. a.). Spart stundenlange manuelle Recherche. | Inspiriert von Ben Aiads Use Case (Lenny's Newsletter) |

## Ausloeser

- „Landingpage erstellen / bauen"
- „Landingpage-Konzept aus Lean Canvas"
- „Conversion-optimierte Seite"
- „Domain-Namen finden"
- „Welche Domain passt zu meinem Projekt?"
