# Website-Building Plugin

Skills fuer den Aufbau von Websites – von der Domain-Namensfindung bis zur fertigen, conversion-optimierten Landingpage.

## Enthaltene Skills

| Skill | Zweck |
|-------|-------|
| **landingpage-builder** | Erstellt vollstaendige, conversion-optimierte Landingpage-Konzepte auf Basis eines Lean Canvas und setzt sie als HTML/CSS/JS-Prototyp um. Nutzt 10-Sektionen-Blueprint, psychologische Frameworks (AIDA, PAS, StoryBrand, Cialdini), CRO-Best-Practices, SEO, rechtliche Pflichtseiten (Impressum, Datenschutz, AGB) fuer DACH, Kontaktformular und Newsletter-Integration. |
| **domain-name-brainstormer** | Generiert kreative Domain-Name-Ideen fuer ein Projekt und prueft die Verfuegbarkeit ueber mehrere TLDs (.com, .io, .dev, .ai u. a.). Spart stundenlange manuelle Recherche. |

## Ausloeser

- „Landingpage erstellen / bauen"
- „Landingpage-Konzept aus Lean Canvas"
- „Conversion-optimierte Seite"
- „Domain-Namen finden"
- „Welche Domain passt zu meinem Projekt?"
