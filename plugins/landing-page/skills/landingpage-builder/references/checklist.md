# Landingpage QA-Checkliste

> Gehe diese Checkliste am Ende des Prozesses mit dem Benutzer durch. Fülle jede Zeile mit einem der drei Status:
> - ✅ Vorhanden & optimiert
> - ⚠️ Vorhanden, aber verbesserbar
> - ❌ Fehlt noch

---

## 1. Struktur & Conversion-Architektur

| # | Prüfpunkt | Status | Anmerkung |
|---|---|---|---|
| 1.1 | **Attention Ratio 1:1** – Keine Navigation, ein Conversion-Ziel | | |
| 1.2 | **10-Sektionen-Reihenfolge** befolgt (Hero → Social Proof → Problem → Lösung → Features → Testimonials → Pricing → FAQ → Finaler CTA → Footer) | | |
| 1.3 | **Message Match** – Headline spiegelt die Traffic-Quelle (Ad, E-Mail, Social) | | |
| 1.4 | **Single CTA** – Alle Buttons führen zum gleichen Conversion-Ziel | | |
| 1.5 | **CTA wiederholt** – Mindestens 2–3× auf der Seite (Hero, nach Features, finaler CTA) | | |
| 1.6 | **Sekundärer CTA** für weniger entschlossene Besucher vorhanden (z.B. Newsletter, Free Guide) | | |
| 1.7 | **Problem vor Lösung** – Problem-Sektion kommt vor der Lösungs-Sektion | | |

## 2. Hero-Bereich (Above the Fold)

| # | Prüfpunkt | Status | Anmerkung |
|---|---|---|---|
| 2.1 | **Headline** beantwortet "Was bringt mir das?" in ≤15 Wörtern | | |
| 2.2 | **Subheadline** ergänzt (wiederholt nicht!) die Headline | | |
| 2.3 | **3-Sekunden-Test** bestanden: Was ist das? Was bringt mir das? Was soll ich tun? | | |
| 2.4 | **Hero-Visual** zeigt das ERGEBNIS, nicht nur das Produkt | | |
| 2.5 | **CTA-Button** im Hero sichtbar ohne Scrollen | | |
| 2.6 | **Mini-Social-Proof** vorhanden (z.B. "5.000+ Nutzer") | | |

## 3. Copywriting

| # | Prüfpunkt | Status | Anmerkung |
|---|---|---|---|
| 3.1 | **Leseniveau Klasse 5–7** – Einfache Sprache, kurze Sätze | | |
| 3.2 | **Benefits vor Features** – Jedes Feature hat einen Nutzen-Satz | | |
| 3.3 | **Du/Sie konsistent** – Eine Anrede durchgängig | | |
| 3.4 | **Spezifische Zahlen** statt vager Versprechen | | |
| 3.5 | **Kein Fachjargon** ohne Erklärung | | |
| 3.6 | **Absätze ≤5 Zeilen** – Scannbar und luftig | | |
| 3.7 | **Gesamtwortanzahl 250–725 Wörter** (ohne rechtliche Seiten) | | |
| 3.8 | **A/B-Test-Varianten** für Headline und CTA vorgeschlagen | | |
